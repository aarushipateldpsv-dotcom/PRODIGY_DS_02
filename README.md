# Task 02: Data Cleaning & Exploratory Data Analysis (EDA)

## 📌 Objective
Perform data cleaning and exploratory data analysis (EDA) to identify patterns, relationships, and trends.

## 📂 Dataset
Sample dataset:  
https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%202

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 🔍 Steps Performed
1. Loaded dataset
2. Data cleaning:
   - Handling missing values
   - Removing duplicates
   - Correcting data types
3. Performed EDA:
   - Summary statistics
   - Correlation analysis
   - Distribution plots
4. Visualizations:
   - Heatmaps
   - Pairplots
   - Histograms

## 📊 Key Insights
- Identified relationships between variables
- Found trends and patterns
- Detected outliers

## 📌 Conclusion
EDA provides a deeper understanding of the dataset and helps in preparing for machine learning models.
